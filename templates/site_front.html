Welcome to the default front page of a stock Nooron instance.

This site is served by Nooron; a product whose aim is to be a distributed container for self-improving knowledge, logic and presentation. Nooron is being released in the spirit of "release early, release often". This is still an early release. The system can currently be considered a dynamic, knowledge-based, web-publishing system. Arbitrary knowledge in, arbitrary XML out. It also has an easily configurable ability to render the XML into more convenient display formats (such as JPEG, Postscript and PDF).
