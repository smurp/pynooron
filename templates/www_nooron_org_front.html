Welcome to Nooron

This site is the home of [and is served by] Nooron: free software which aims to be a distributed container for evolving knowledge, logic and presentation. Nooron is being released early and often.

,

Current Version

This version (nooron-0.2.2) can be considered a knowledge publishing system. Knowledge in, documents out. The knowledge must currently be in an unusual format (.pykb), but any new PyOKBC backends will work with Nooron. It will be relatively easy to write backends for many kinds of knowledge source including (but not limited to): RDF, TopicMaps, Cyc, PostgreSQL, MySQL, Oracle, Bibtex, IMAP, LDAP, WordNet, PDB, CSV, CVS and nearly every other structured source. Likewise, Nooron can generate basically any kind of output; HTML, XML and other similar formats directly via 'NooronPageTempates' (NPT, a slight mutation of ZPT) or thereafter into anything that a command line utility can generate from the NPT output. At the moment that includes: GraphViz .dot files, Docbook, Postscript, PDF and JPG, but it is completely open-ended because the whole transformer system is knowledge driven.
